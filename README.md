# ✨ AI Poem Generator

Transform your thoughts into beautiful poetry with the power of AI.

## 🌐 Live Demo

🔗 https://aipoemgeneratorr.netlify.app/

## 📖 About The Project

AI Poem Generator is a modern AI-powered web application that generates creative poems based on user prompts. Users can choose different poetry styles, switch between dark and light themes, and enjoy a beautiful glassmorphism-inspired interface.

## 🚀 Features

* ✨ AI-powered poem generation
* 🎭 Multiple poetry styles

  * ❤️ Romantic
  * 🌿 Nature
  * ✨ Inspirational
  * 😔 Sad
* 🌙 Dark / Light mode toggle
* 📋 One-click poem copy
* 🎨 Modern Glassmorphism UI
* 📱 Fully Responsive Design
* ⚡ Fast and Interactive Experience

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)
* Axios
* SheCodes AI API
